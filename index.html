001
F1 Telemetry Anomaly Detection System

An end-to-end ML pipeline for detecting anomalous laps in Formula 1 telemetry, built on the OpenF1 API. Six Python stages handle session fetching, IQR-based statistical detection, telemetry collection, and XGBoost model training across 14 signals. A Streamlit dashboard adds a predictive layer: rolling 0–100 degradation risk scoring and a lap-by-lap replay slider simulating live anomaly probability.
